# TaxPal Tailwindcss

## Overview

TaxPal Landing page

## Built with

+ Tailwindcss
+ JavaScript

## Development

1- Clone the repository and change directory.
```
git clone https://github.com/mujahedjs/taxpal-tailwindcss.git
cd taxpal-tailwindcss
```
2- Install npm dependencies
```
npm install -g tailwindcss
```
3- Run app locally
```
npm run build
npm run watch
```
